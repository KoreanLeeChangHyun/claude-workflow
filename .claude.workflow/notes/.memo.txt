wf = workflow 

OPEN -> PROGRESS -> REVIEW -> DONE

wf -o     : 새 티켓 후 Prompt 작성    티켓은 OPEN
wf -p N   : 티켓 Prompt 작성          티켓은 OPEN
wf -s N   : 티켓 워크플로우 제출      티켓은 PROGRESS (진행 중) -> REVIEW (진행 완료) 
wf -d N   : 티켓 종료                 티켓은 DONE
wf -c N   : 티켓 삭제                 티켓 삭제 보드 에서도 삭제


wf -p N   : 티켓 Progress 로 수동 전환 (기본은 워크플로우가 자동 결정)
wf -r N   : 티켓 Review 로 수동 전환 (기본은 워크플로우가 자동 결정)
