# AT-06: Goal-Context Purpose Alignment

## Evaluation Dimension

Assess whether the `goal` and `context` fields describe the same underlying
purpose — i.e., the true objective stated in context is the same as the
achievement described in goal.

## Scoring Criteria

| Score | Verdict | Criterion |
|-------|---------|-----------|
| 5     | PASS    | Goal and context describe the same objective; context elaborates on goal without introducing a different purpose |
| 4     | PASS    | Minor terminology difference between goal and context but clearly the same purpose |
| 3     | WARN    | Context hints at a secondary purpose or unstated objective that partially differs from goal |
| 2     | FAIL    | Context reveals a significantly different objective than what goal states |
| 1     | FAIL    | Context describes a completely different or contradictory purpose compared to goal |

**Note (Hard Gate):** This item is a hard gate. A FAIL score (≤ 2) forces the
entire Tier-2 verdict to FAIL immediately, overriding all other item scores and
the weighted average. The `hard_gate_failed` list will include `"AT-06"`.

## Bias Mitigation

Swap goal/context order and re-evaluate; take the average of both evaluations
to reduce positional bias. Use a cross-family model when available.

context says "the real purpose is infrastructure capacity validation" — these
are different objectives.

purpose described in the context?"
