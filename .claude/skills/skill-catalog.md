| 명령어 | 자동 로드 스킬 | 용도 |

| implement | review-code-quality, devops-verification | 코드 품질 검사(Generator-Critic 루프 포함), 완료 전 검증(점진적 검증 포함). 에셋 관리 키워드 감지 시 매니저 스킬 조건부 로드 |
